Module `predict_country` pour prédire le pays d'un document.
 
Ce module utilise un modèle RandomForest pré-entraîné sur des vecteurs TF-IDF
pour déterminer le pays associé à chaque document. Il charge le modèle et son
encodeur de labels, applique les prédictions sur les vecteurs TF-IDF, et
sauvegarde les résultats dans un fichier CSV avec toutes les probabilités pour
chaque classe.
 
Fonctionnalités principales:
- Chargement du modèle RandomForest et de l'encodeur de labels.
- Chargement des vecteurs TF-IDF des documents.
- Vérification et alignement des features avec le modèle.
- Prédiction des probabilités pour chaque pays.
- Application d'un seuil pour filtrer les prédictions incertaines.
- Export des résultats avec toutes les probabilités pour debug.

Functions
       
predict_country(threshold: float = 0.2) -> None
Prédit le pays à partir de vecteurs TF-IDF.
 
Paramètres:
----------
threshold : float, optionnel (default=0.2)
    Seuil minimal de probabilité pour accepter une prédiction. Si la
    probabilité maximale pour un document est inférieure à ce seuil,
    le pays sera marqué comme "unknown".
 
Résultats:
----------
- Un fichier CSV dans `output/predictions/tfidf_vectors_with_country_predictions.csv`
  contenant les colonnes:
    - doc : identifiant du document
    - predicted_country : pays prédit ou "unknown"
    - confidence : probabilité de la prédiction
    - proba_<pays> : probabilité pour chaque pays
