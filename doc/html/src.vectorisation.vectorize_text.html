Module de vectorisation de texte en vecteurs TF-IDF.
 
Ce module prend une liste de documents et leurs textes correspondants,
applique une vectorisation TF-IDF en anglais, et sauvegarde le résultat
dans un fichier CSV pour un usage ultérieur, par exemple pour la
prédiction de domaine ou d'autres tâches de NLP.

Functions
       
vectorize_text(docs, texts, max_features=300)
Vectorise les textes en utilisant TF-IDF et sauvegarde les vecteurs.
 
Parameters
----------
docs : list of str
    Liste d'identifiants pour chaque document.
texts : list of str
    Liste des textes correspondants à vectoriser.
max_features : int, optional
    Nombre maximal de features à retenir dans le TF-IDF (par défaut 300).
 
Returns
-------
tuple
    - pd.DataFrame : DataFrame contenant les vecteurs TF-IDF et la colonne 'doc'.
    - TfidfVectorizer ou None : L'objet vectorizer utilisé, ou None si aucun texte fourni.
 
Notes
-----
- Les vecteurs sont sauvegardés dans OUTPUT_CSV au format CSV avec séparateur ';'.
- Les colonnes correspondent aux termes extraits par le TF-IDF.
- Si aucun texte n'est fourni, la fonction retourne un DataFrame vide.
