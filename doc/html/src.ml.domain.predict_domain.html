Module de prédiction de domaine à partir de vecteurs TF-IDF.
 
Ce module charge un modèle de classification (Logistic Regression)
et un encodeur de labels, applique les vecteurs TF-IDF pour prédire
le domaine de chaque document, et sauvegarde les résultats avec
les scores de confiance. Il est possible d'utiliser un seuil pour
retourner 'unknown' lorsque la prédiction est incertaine.

Functions
       
predict_domain(threshold: float = None)
Prédit le domaine pour chaque document à partir de vecteurs TF-IDF.
 
Parameters
----------
threshold : float, optional
    Seuil de confiance (entre 0 et 1). Si la probabilité maximale pour
    un document est inférieure à ce seuil, la prédiction sera 'unknown'.
 
Notes
-----
- Aligne les colonnes des vecteurs sur celles attendues par le modèle.
- Sauvegarde les résultats dans un fichier CSV avec :
    - doc : identifiant du document
    - predicted_domain : domaine prédit
    - confidence_score : score de confiance de la prédiction
